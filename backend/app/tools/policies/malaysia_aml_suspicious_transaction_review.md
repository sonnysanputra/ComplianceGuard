# Suspicious Transaction Review

A case must be reviewed when a transaction appears unusual compared to the
reporting institution's knowledge of the customer, or when the economic purpose
or legality of the transaction is unclear.

## System Action

The AI system must:
- review the customer profile;
- review the transaction pattern;
- check watchlist and sanctions indicators;
- retrieve the relevant policy;
- prepare an investigation summary;
- recommend escalation where suspicion remains.

The AI system must not automatically submit a Suspicious Transaction Report. A
human compliance analyst must review the case before any submission.
